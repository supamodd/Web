Дескрипторы физического форматирования

В HTML всё делается через дескрипторы.

<p> Параграф ― этот дескриптор описывает абзац.
<b> (Bold) ― этот дескриптор делает шрифт жирным.
<i> (Italic) ― этот дескриптор делает шрифт наклонным.
<u> (Underline) ― этот дескриптор делает шрифт подчеркнутым.
<s> (Strikeout) ― этот дескриптор делает шрифт зачеркнутым.
<tt> (Teletype text) ― этот дескриптор делает шрифт моноширным или монохромным. Каждый символ занимает одинаковую площадь на экране.(DEPRECATED)
<sub> (Subscript) ― этот дескриптор делает шрифт внизу.
<sup> (Superscript) ― этот дескриптор делает шрифт наверху.
Дескрипторы логического форматирования

Дескрипторы логического форматирования нужны для анализа страницы поисковыми машинами. Многие из них замещают дескрипторы физического форматирования.

<strong> (Bold) ― этот дескриптор делает шрифт жирным.
<em> (Emphasis) ― этот дескриптор делает шрифт курсивным.
<strike> (Strike - Delete) ― этот дескриптор делает шрифт зачеркнутым.
<code> (Code) ― этот дескриптор делает шрифт моноширным.
<mark> (Mark) ― этот дескриптор делает шрифт выделенным.
<code> (Citation) ― этот дескриптор делает шрифт цитирование.
